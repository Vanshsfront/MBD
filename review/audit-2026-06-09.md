# MBD Clinic OS — Universal Audit (2026-06-09)

Branch: `audit-fixes/2026-06-06` merged to `main`. Target: https://mbd-y24p.vercel.app/. 13 dimensions assessed (1, 3, 4, 5, 8, 9, 10, 11, 13, 15, 18, 19, 20). PHI in scope; HIPAA-aligned marketing claim live; DPDPA 2023 in force.

---

## Executive Summary

The audit branch shipped two real wins: a server-stamped `Staff.sessionVersion` revocation kill-switch and idempotency keys on `/api/payments` + `/api/invoices`. Bcrypt cost is 12, TLS posture is clean (TLS 1.3 + AEAD, 2-year HSTS preload), Brotli served, no `dangerouslySetInnerHTML`, all Prisma queries are typed (no SQLi sink), license posture is unblocked (no AGPL/SSPL/GPL contamination), and there is no third-party tracking surface.

**Critical exposure (24h SLA):** `xmldom@0.6.0` carries 7 advisories incl. CVSS 9.8; reached at runtime by every DOCX render path (consent, consultation, invoice). Mitigation in `package.json` overrides pins to a still-vulnerable version. Drop `docxtemplater-image-module-free` or replace with maintained module before next deploy.

**High exposure (72h SLA):** (a) `next@16.2.5` has a middleware-bypass advisory (GHSA-26hh-7cqf-hhc6) — patch to 16.2.7. (b) The standard sign-out never bumps `sessionVersion`; stolen JWTs survive logout for the 8h window. (c) Password floor is 6 chars (`mbd2026` is 7 chars and the seeded shared default) AND the in-process rate limiter is sharded across Vercel Lambda instances — verified: 15 wrong-password attempts to /api/auth/callback/credentials produced zero 429s. Credential stuffing is effectively unthrottled. (d) RLS is disabled on all 35 `public.*` tables; the Supabase Realtime channel authenticates with only the public anon key and broadcasts cross-centre on `postgres_changes` (dormant only because the publication is empty). (e) PHI stored cleartext though `pgcrypto` + `supabase_vault` are installed and unused. (f) No CI pipeline, no automated tests, no SBOM, no MFA for OWNER/ADMIN, no incident runbook, no on-call. (g) Indian patient PHI is hosted on AWS Singapore with no DPA, no residency notice, and a public "HIPAA aligned" claim with no BAA evidence.

**14-day cleanup:** rotate `AUTH_SECRET`/`DATABASE_URL`/`SUPABASE_SERVICE_ROLE_KEY` after replacing `mbd2026`; add per-request `X-Request-ID` + audit-log writers for LOGIN/EXPORT events; replace CUID-default tokens on `IntakeToken`/`ClientPortalToken` with `randomBytes(32)`; close the open redirect at `/login?from=`; remove `'unsafe-inline'` from CSP via per-request nonce; migrate `node-cron` jobs to Vercel Cron (data-retention is currently theatre); ship privacy.md + cookie policy + grievance officer.

The audit-fixes branch correctly identified the right scaffolding to add but consistently stopped one step short of wiring it through the request lifecycle. The structural fixes are cheap; the credential/secrets hygiene is the urgent piece.

## Findings

Findings below are excerpted at the metadata level; full diffs, acceptance criteria, and regression tests are preserved in the source dimension reports. Each record carries: ID · severity · title · location · root cause · primary remediation.

### Dimension 1 — Authentication & Session Security

**AUTH-003 · MEDIUM · 8-hour single JWT with no AT/RT split** — `src/lib/auth.ts:66` sets `maxAge: 8*60*60`; no sliding window. Add `updateAge: 15*60` and a 60-min idle gate against `token.iat`.

**AUTH-007 · HIGH · Default sign-out does not increment sessionVersion** — `src/components/layout/dashboard-shell.tsx:105-108` calls `signOut()` only; `Staff.sessionVersion` is never bumped, so stolen JWTs survive logout for the 8h window. Wire `prisma.staff.update({ data: { sessionVersion: { increment: 1 } } })` inside the server action before the `signOut()` call.

**AUTH-009 · HIGH · Min password length 6** — `src/app/api/profile/password/route.ts:13` + `src/app/api/admin/staff/route.ts:29,46` all use `z.string().min(6)`. UI hint accepts the seeded `mbd2026`. Lift to 12-char floor + blocklist. Pairs with CRYPT-001.

**AUTH-010 · HIGH · Rate limiter sharded across Lambdas** — `src/lib/rate-limit.ts:39` is `Map`-backed. Live test (2026-06-09): 15 wrong-password attempts to `/api/auth/callback/credentials` produced zero 429s. Migrate to Upstash Redis / Vercel KV.

**AUTH-011 · MEDIUM · No self-service password reset** — only admin-mediated `PATCH /api/admin/staff { resetPassword }` exists. Add `POST /api/auth/forgot-password` with 256-bit sha256-stored token, 60 min TTL, single-use.

**AUTH-013 · HIGH · No MFA** — `src/lib/auth.ts:68-113` only registers Credentials. `/api/auth/providers` confirms. Add TOTP for OWNER/ADMIN/DEV/CONSULTANT roles.

**AUTH-014 · MEDIUM · No step-up auth for role change / staff deletion** — `src/app/api/admin/staff/route.ts:99-170` mutates with only `requirePermission('admin:manage_staff')`. Introduce single-use 5-min step-up tokens; require `X-Step-Up-Token` header on staff PATCH/DELETE and revoke-sessions.

### Dimension 3 — Cryptography & Secrets Management

**CRYPT-001 · MEDIUM · Hardcoded `mbd2026` bootstrap password** — `prisma/seed.ts:27`, `src/components/admin/staff-dialogs.tsx:147`, `scripts/capture-screenshots.ts:45`. Replace with `randomBytes(18).base64url`; add `Staff.mustResetPassword` column.

**CRYPT-002 · LOW · `mbd2026` in git history; no entropy scanner** — Add `.github/workflows/security-scan.yml` running gitleaks + trufflehog on PR; rotate AUTH_SECRET/DATABASE_URL/SUPABASE_SERVICE_ROLE_KEY after CRYPT-001.

**CRYPT-004 · MEDIUM · Secret rotation requires redeploy; no kid versioning** — `src/lib/auth.ts`. Adopt dual-secret acceptance (`secret: [current, previous]`); document runbook at `docs/security/secret-rotation.md`.

**CRYPT-007 · HIGH · PHI stored cleartext; pgcrypto/Vault installed but unused** — `prisma/schema.prisma` `Client.dob` (line 222), `Consultation.vitals`/`comorbidities`/`diagnosis` (lines 333-341), `MedicalHistory.*`, `IntakeForm.data`. Add `src/lib/crypto.ts` with AES-256-GCM envelope; rename columns to `*Enc`; Prisma middleware enforces on every write.

**CRYPT-008 · HIGH · CUID v1 PHI-access tokens** — `prisma/schema.prisma:273` (ClientPortalToken.token), `:290` (IntakeToken.token). Drop `@default(cuid())`; set explicitly from `randomBytes(32).base64url` in `src/lib/tokens.ts`; revoke all existing tokens.

**CRYPT-010 · MEDIUM · Backup posture unverified** — No `pg_dump` scripts, no Vercel cron, no documented retention. Add `docs/security/backup-and-retention.md`; replace node-cron with Vercel Cron + Supabase Edge Function for retention purges.

### Dimension 4 — Input Handling, Injection & Output Encoding

**INJ-005 · MEDIUM · CSP `'unsafe-inline'`** — `next.config.ts:27-34`. Live header on /login confirms `script-src 'self' 'unsafe-inline'; style-src 'self' 'unsafe-inline'`. Move to per-request nonce via middleware (`'nonce-…' 'strict-dynamic'`). Folds into LEGAL-006 for header-tier remediation.

**INJ-010 · MEDIUM · SVG signature; attachments no magic-byte check** — `src/app/api/profile/signature/route.ts:11` `startsWith('data:image/')` accepts `data:image/svg+xml`. `src/app/api/consultations/[id]/attachments/route.ts:102` trusts `file.type`. Restrict signature regex to `image/(png|jpe?g)`; add 8-byte magic-byte sniff for PDF/DOCX.

**INJ-012 · MEDIUM · XML parsers not hardened (xmldom via docxtemplater-image-module-free, ExcelJS)** — Add DOCTYPE/ENTITY guard before `wb.xlsx.load()` and before storage write; migrate to `@xmldom/xmldom@0.9.10` via overrides; pairs with SUPPLY-001 for upstream drop.

**INJ-014 · MEDIUM · Open redirect at `/login?from=`** — `src/app/(auth)/login/login-form.tsx:20,39` calls `router.push(callbackUrl)` with no allow-list. Sanitize: only accept values matching `/^\/[^\/\\]/`; reject protocol-relative + percent-encoded protocol smuggling.

### Dimension 5 — Dependency & Supply Chain Security

**SUPPLY-001 · CRITICAL · xmldom@0.6.0 + next@16.2.5 + tmp** — `npm audit`: 1 critical, 2 high, 9 moderate. `xmldom` has 7 advisories (CVSS up to 9.8) and `fixAvailable:false`; override pins to a vulnerable version. Drop `docxtemplater-image-module-free` (the only consumer) — either purchase `@docxtemplater/image-module` (no xmldom) or refactor `src/lib/templates/docx.ts` to inject images via direct zip writes. Bump `next` to 16.2.7 (GHSA-26hh-7cqf-hhc6 middleware bypass).

**SUPPLY-002 · MEDIUM · No CI enforces `npm ci`** — Add `vercel.json` `installCommand: "npm ci"` + `.github/workflows/ci.yml` running `npm ci && npm run lint && npm run typecheck && npm audit --audit-level=high --omit=dev`.

**SUPPLY-003 · LOW · All 90 deps use caret; next-auth on beta** — Add `.npmrc save-exact=true`; pin `next-auth`, `bcryptjs`, `pg`, `@prisma/client`, `prisma`.

**SUPPLY-004 · LOW · Dependabot ≤24h old, no daily security cadence** — Add second updates block with `interval: daily` for security-only PRs.

**SUPPLY-005 · LOW · engines.node ≥18.17 permits EOL** — Bump to `">=22.0.0"`; add `.nvmrc`.

**SUPPLY-009 · MEDIUM · No SBOM produced per release** — Add `"sbom": "cyclonedx-npm --output-format JSON --output-file sbom.json --omit dev"`; archive artifact per release SHA for 365 days.

### Dimension 8 — Observability, Monitoring & Incident Response

**OBS-001 · MEDIUM · Erasure `reason` logged cleartext; AuditLog stores raw IP/UA for 7 yrs** — `src/app/api/clients/[id]/erase/route.ts:117-126` + `prisma/schema.prisma:784-785`. Hash UA/IP in AuditLog; replace free-text reason with enum + short-retention table.

**OBS-002 · MEDIUM · `childForRequest()` is dead code** — `src/lib/logger.ts:82-86` mints requestId; zero call sites in `src/app`. Wire via `withRequestLogger` wrapper; AuditLog gains `requestId` column.

**OBS-004 · HIGH · No SLO/SLI, no error/saturation/traffic monitoring, no alerts** — No Sentry/Datadog/OTel/Honeycomb dep. RumEvent captures client-side latency only. Add Sentry or Vercel Log Drain → Better Stack/Axiom; define SLOs for the four golden signals; route burn-rate alerts to pager.

**OBS-005 · HIGH · `LOGIN` audit-log never written** — `src/lib/auth.ts:75-112` returns null on every failure path without `createAuditLog`. `AuditAction = 'LOGIN'` has no writer. Add LOGIN audit rows with `result: 'ok'|'wrong_password'|'rate_limited'|'unknown_or_inactive'`; cron-fed anomaly detector.

**OBS-006 · MEDIUM · No distributed tracing; X-Request-ID not echoed** — Live probe with `X-Request-ID: test-trace-id-123` not echoed back. Add `@vercel/otel` or wire `childForRequest()` plus response header echo.

**OBS-007 · HIGH · No incident response runbook; no DPDPA §8(6) 72h breach notification** — Add `reference/incident-runbook.md` covering Triage → Contain → Notify (DPB India, HIPAA §164.404, affected principals) → Postmortem.

**OBS-008 · HIGH · No on-call rotation, no paging system** — Add `reference/oncall.md` with primary/secondary by week; wire Opsgenie/PagerDuty/Twilio webhook to OBS-004 alerts; quarterly synthetic drill.

### Dimension 9 — API Design, Performance & Resilience

**API-001 · HIGH · In-process limiter on serverless; mutations unprotected** — `/api/payments`, `/api/invoices`, `/api/appointments`, `/api/clients/[id]` have no `consume()` at all. Live test (2026-06-09): 130 rapid POSTs to /api/rum (limit 120/min/IP) all returned 200. Migrate to `@upstash/ratelimit` + Redis; expose `X-RateLimit-*` on all responses, not just 429s.

**API-003 · MEDIUM · No client-driven pagination; several endpoints have no `take`** — `/api/appointments`, `/api/attendance`, `/api/reports/mis-csv`, cron jobs. Clamp date windows to 90 days; add cursor-based pagination; stream MIS CSV.

**API-004 · MEDIUM · Slow query logging off in prod; no app middleware** — `pg_settings.log_min_duration_statement = -1`; Prisma client built without `log:['query']`. Request Supabase `log_min_duration_statement = 500ms`; add `$on('query')` middleware emitting `event: 'slow_query'` above 500ms.

**API-008 · MEDIUM · No request body size limit; uploads buffered before size check** — `next.config.ts` has no `bodySizeLimit`. `/api/consultations/[id]/attachments` checks `MAX_BYTES = 25MB` only after `formData()` resolves. `/api/rum` is unauthenticated POST with no Content-Length gate. Add `assertBodySize()` helper before `req.json()`/`formData()`; declare `experimental.serverActions.bodySizeLimit` explicitly.

### Dimension 10 — Real-Time Connections & Event Streaming
